# Summary

## title

友だち紹介キャンペーン用コードを発行する

## blogUrl

https://solution.karte.io/blog/2026/06/generate-referral-link/

## description

user_idを受け取り、UUIDベースの紹介コード（referral_code）を生成してKARTEイベントとして記録するHTTPエンドポイントです。生成した紹介コードをレスポンスで返します。呼び出し元でreferral_codeをURLに付与して紹介リンクとして使用します。

## category

Craft Functions,KARTEイベント,CRAFT_ENDPOINT

http
