KARTE Dataubのジョブフローの実行時に指定したGCSバケット上のファイルを削除する

KARTE Datahubジョブフロー実行時に、指定したGCSバケット上のファイルを削除します。削除対象のGCSバケットのプロジェクト名、バケット名、削除対象ファイルパスはDatahubクエリで指定します。
