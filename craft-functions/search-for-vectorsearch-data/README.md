# Summary

## title

Craft Vector Searchのベクトルデータを検索する

## blogUrl

https://solution.karte.io/blog/2026/02/semantic-search-with-vector-search/

## description

HTTPリクエストで受け取った検索クエリ文字列をCraft AI Modules経由でベクトル化し、そのベクトルデータを使ってCraft Vector Searchのデータを検索します。namespaceによるフィルタリング、マルチモーダル、キーワード検索にも対応しています。
