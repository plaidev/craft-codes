# Summary

## title

MCP経由でCraft RAG上のコンテンツを利用可能にする

## blogUrl

https://solution.karte.io/blog/2026/03/build-mcp-server-with-functions/

## description

MCPサーバーを立ち上げて、MCPクライアントからCraft RAG上のコンテンツを利用可能にするためのテンプレートです。指定したCraft RAGコーパスから類似コンテンツを検索し、LLMを利用して質問に対する回答を生成します。認証にはCraft Secret Managerからトークンを取得し、セキュアなアクセスを実現します。
