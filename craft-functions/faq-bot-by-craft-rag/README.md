# Summary

## title
Craft RAGを活用し、KARTE Talk のFAQチャットボットを構築する。

## blogUrl
https://solution.karte.io/blog/2026/03/faq-bot-by-craft-rag/

## description

Craft RAG に登録されたFAQデータをもとに、KARTE Talk のユーザーメッセージへ自動回答するBOTを構築するためのテンプレートです。
ユーザーの質問とFAQのベクトル類似度検索により関連コンテンツを取得し、Gemini で自然な回答を生成します。
関連FAQが見つからない場合は、有人オペレーターへの切り替えも可能です。
