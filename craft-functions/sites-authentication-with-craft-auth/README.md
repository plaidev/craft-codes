# Summary

## title

Craft Authを使った認証付きカスタム管理画面

## blogUrl

https://solution.karte.io/blog/2026/02/build-admin-console-with-craft-auth-and-sites/

## description

Craft Authを使って認証したユーザーだけが閲覧可能なサイトのバックエンドファンクションです。同一エンドポイントでCraft Authによるサインインと署名付きCookieの発行、およびユーザーロールに応じたダミーコンテンツの返却を提供します。リクエストボディのactionパラメータで処理を分岐し、特定のロールを保持するユーザーのみにコンテンツを返却します。
