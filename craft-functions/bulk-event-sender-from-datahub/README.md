# Summary

## title
Datahubと連携し、KARTEイベントを一括送信する。

## blogUrl
https://solution.karte.io/blog/2026/01/bulk-event-sender-from-datahub/

## description

オフライン店舗での購買データや、他サービスとの連携データなどをKARTEイベントとして一括送信するための、Datahub × Craft Functions を活用したテンプレートです。
接客サービスのゴール設定・イベントを使わないと設定しにくい条件でのセグメント作成などにご活用いただけます。
