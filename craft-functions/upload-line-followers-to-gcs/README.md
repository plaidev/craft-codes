# Summary

## title

LINE公式アカウントの友だち追加したユーザーのIDリストをGoogle Cloud Storageに保存する

## blogUrl

https://solution.karte.io/blog/2026/06/upload-line-followers-to-gcs

## description

LINE公式アカウントのGet Follower IDs APIを使用して友だち追加したユーザーのIDを取得し、JSONL形式でGoogle Cloud Storageに保存します。保存後にDatahubジョブフローを実行することで、LINEのユーザーリストをKARTE Datahubなどに取り込めます。
