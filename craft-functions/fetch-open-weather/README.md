# Summary

## title

OpenWeatherMap から取得した天気情報を含めて KARTE にイベント送信する

## blogUrl

https://solution.plaid.co.jp/craft-weather-api

## description

このサンプルコードは、ユーザーの位置情報を元に OpenWeatherMap から天気情報を取得し、 KARTE にイベント送信します。取得した天気情報は、KARTE で配信するアクションの出し分けなどに利用できます。
