# Summary

## title

Craft Cross CMSコンテンツのCraft AI Modulesによる自動翻訳

## blogUrl
https://solution.karte.io/blog/2026/06/xcms-auto-translate-aimodules/

## description

Craft Cross CMSで翻訳リクエストフラグを立てて保存した際に、Craft AI Modules（Gemini）を使って自動的に指定した言語に翻訳し、同一コンテンツ内の言語別フィールド（例: title_en, body_zh）に翻訳結果を書き込みます。複数言語への同時翻訳やリッチテキストの構造保持にも対応しています。
