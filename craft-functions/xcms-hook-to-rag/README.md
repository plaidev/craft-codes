# Summary

## title

Craft Cross CMSのコンテンツ変更をCraft RAGに同期する

## blogUrl

https://solution.karte.io/blog/2025/10/xcms-hook-to-rag/

## description

Craft Cross CMSでコンテンツの公開/非公開が行われた際に、自動的にCraft RAGにコンテンツを同期します。CMS側のHookトリガーを使用して、指定されたモデルのコンテンツ変更を検知し、CMSからコンテンツを取得、Craft SitesへのアップロードとCraft RAGへのインポートを行います。

## category

Craft Functions,Craft Cross CMS,Craft RAG,Craft Sites
