# Summary

## title

Craft Document DBで商品カタログを検索する

## blogUrl

https://solution.karte.io/blog/2026/04/docdb-introduction/

## description

Craft Document DBに登録された商品データを、カテゴリ・価格帯・タグなどの条件で検索するAPIです。範囲検索（gte/lte）、配列フィールドの検索（has）、カーソルベースのページネーションに対応しています。フロントエンドから受け取った検索条件をDocument DBのクエリに変換して実行し、結果を返します。
